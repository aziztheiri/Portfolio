Magharebia Insurance Website Renovation

This project represents a complete renovation of the Magharebia Insurance Website, built with a robust microservices architecture using Spring Boot for backend services and Angular for the frontend. The goal was to modernize the platform, enhance scalability, and streamline complex operations within the insurance domain.

Key Features:

  • User Management Service: Developed a comprehensive user management module that handles registration, authentication, profile management, password reset, and account activation/deactivation. Integrated with external identity providers for enhanced security.
  • Microservices Integration: Leveraged RESTful APIs and RabbitMQ for asynchronous communication between services, ensuring reliable and scalable inter-service operations.
  • Enhanced Security: Implemented industry-standard security practices including BCrypt for password hashing, token-based validations, and multi-channel notifications (email and SMS) for critical account actions.
  • AI-Powered Client Segmentation: Designed and deployed an AI model to segment clients using advanced clustering and classification algorithms. This model tested multiple algorithms to select the most efficient strategy for predicting client clusters, enabling personalized insurance offerings.
  • Cloud Integration: Integrated Cloudinary for efficient image management and leveraged cloud services to enhance system performance and scalability.

Technologies: Spring Boot, Angular, RabbitMQ, Keycloak, BCrypt, Cloudinary, AI/ML (clustering and classification)

Hotel Management System

The Hotel Management System is a comprehensive platform designed to streamline hotel operations and enhance guest experiences. Built using Spring Boot and Angular , this system efficiently manages room listings, user authentication, and booking processes, ensuring smooth operations from both administrative and customer perspectives.

Key Features:

  • Room Management: Empower administrators with full CRUD capabilities to manage room details, including names, types, pricing, and availability. Integrated with Cloudinary for efficient image uploads and storage.
  • User Authentication: Secure user registration and login processes, featuring role-based access control and encrypted passwords using BCrypt.
  • Reservation System: Seamless booking functionality that handles reservations, calculates pricing based on stay duration, and updates room availability in real time.
  • Administrative Tools: Provides robust dashboards for monitoring reservations, updating room statuses, and managing user profiles to ensure optimal operational control.

Technologies: Spring Boot, Spring Data JPA, Cloudinary, BCrypt, RESTful APIs, JWT.
